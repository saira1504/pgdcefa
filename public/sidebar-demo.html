🚀 Sidebar Colapsable - Demo

Este es un ejemplo de cómo funciona el sidebar colapsable. Haz clic en el botón hamburguesa (tres rayas) en el footer del sidebar para colapsarlo y expandirlo.

Colapsable

El sidebar se puede contraer para mostrar solo los iconos, ahorrando espacio en pantalla.

Persistente

El estado del sidebar se guarda en localStorage y se restaura al recargar la página.

Tooltips

Cuando está colapsado, aparecen tooltips al hacer hover sobre los iconos.

Responsive

Se adapta automáticamente a diferentes tamaños de pantalla.

Personalizable

Fácil de personalizar con variables CSS y estilos modulares.

Rápido

Transiciones suaves y animaciones optimizadas para una mejor experiencia de usuario.

📱 Características del Sidebar

  • Toggle Button: Botón hamburguesa en el footer del sidebar
  • Estado Colapsado: Solo muestra iconos (70px de ancho)
  • Estado Expandido: Muestra iconos + texto (280px de ancho)
  • Tooltips: Aparecen al hacer hover en estado colapsado
  • Persistencia: Recuerda el estado entre sesiones
  • Responsive: Se adapta a dispositivos móviles
  • Transiciones: Animaciones suaves al cambiar estados
  • Botón en Footer: No interfiere con la información del usuario

🎯 Cómo Usar

  1. Busca el botón toggle: Está en el footer del sidebar (abajo)
  2. Haz clic: El sidebar se colapsará mostrando solo iconos
  3. Vuelve a hacer clic: Se expandirá mostrando iconos + texto
  4. Hover en iconos: Cuando esté colapsado, aparecen tooltips
  5. Estado persistente: Se recuerda al recargar la página
